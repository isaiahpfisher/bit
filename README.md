This is a basic implementation of Git, written in Python. It is not intended to be a complete implementation, but rather a simple proof of concept for the sake of learning and developing a better mental model of how Git works on the inside.

## Supported Commands

- init
- clone
- config
- add
- rm
- status
- stash
- commit
- log
- branch
- checkout
- diff
- merge
- reset
- restore
